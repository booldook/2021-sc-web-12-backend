# node 프로젝트 시작하기
## 1. node프로그램 설치
  - [nodejs.org](https://nodejs.org) 에서 설치한다.
## 2. 다음과 같이 node와 npm 이 설치되었는지 확인한다.
```bash
node -v
npm -v
```
## 3. 프로젝트 폴더를 만들고 폴더의 터미널 창을 열어서 다음과 같이 시작한다.
```bash
c:/node-sample>  node init -y
```
## 4. 필요한 모듈들을 npmjs에서 검색하여 설치한다.
```bash
npm i express moment
```
